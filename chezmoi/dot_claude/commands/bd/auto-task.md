このセッションは `beads-loop` により起動されています。
User Input の指示にしたがってタスクを進行してください。

プログラムから起動されているため、ユーザーに質問を行なうことはできません。beads のタスク管理を活用し自律的に進行してください。

<system-reminder>
コンディションの不整合等により完了済みのタスクが resume される可能性があります。タスクが実行済みかつ `feature-review:ng` ラベルも付与されていない場合はそのまま終了してください。
</system-reminder>
